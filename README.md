# Handcrafted Haven

**Handcrafted Haven** is a full-stack web application designed as a virtual marketplace for artisans and crafters. It connects creators with consumers who value high-quality, sustainable, and unique handmade products.

## Project Overview
- **Course:** WDD 430 - Web Full-Stack Development
- **Institution:** BYU-Idaho
- **Developer:** Bryan Wesley Santana

## Technology Stack
- **Frontend:** Next.js, Tailwind CSS
- **Backend:** Node.js
- **Deployment:** Vercel
- **Project Management:** GitHub Projects (Kanban)

## Design System
- **Primary:** Dark Slate (#111827)
- **Background:** Soft Beige (#F9FAFB)
- **Accent:** Muted Green (#4D7C0F)
- **Typography:** Montserrat & Inter

## 📝 Key Features
- Artisan Profiles & Storytelling
- Curated Product Listings
- Advanced Search and Filtering
- User Reviews and Ratings
- Mobile-First Responsive Design
